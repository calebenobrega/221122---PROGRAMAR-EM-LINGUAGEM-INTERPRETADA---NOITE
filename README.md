# Programar em Linguagem Interpretada
Projeto simples da disciplina com atividades praticas em JavaScript.

Objetivo: treinar logica, estruturas de controle, funcoes e manipulacao de dados.

Este repositorio esta organizado em duas partes principais:

- Entrega fase 1: projeto principal da turma, com foco no sistema de estoque (CRUD).
- Exercicios: lista de exercicios separados por pastas (Exercicio 1, Exercicio 2 e Exercicio 3) para praticar conceitos da materia.

A IA foi usada apenas como apoio para formatacao do codigo, design e comentarios de ajuda.
